Here you can see the properties that you can configure for the LDAP plugin of the Camunda BPM process engine.

## allowAnonymousLogin 

<dl>
  <dt>Name:</dt>
  <dd>
    <code>
      allowAnonymousLogin 
    </code>
  </dd>
  <dt>
    Required:
  </dt>

  </dd>
  <dt>
    Related to:
  </dt>
  <dd>
    OTHER PROPERTIES THAT ARE RELATED TO THIS ONE
  </dd>
  <dt>
    Description
  <dt>
